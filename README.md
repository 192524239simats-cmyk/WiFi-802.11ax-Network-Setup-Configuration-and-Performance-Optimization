# WiFi (802.11ax) Network Setup, Configuration, and Performance Optimization

## Project Overview
This project focuses on the design, setup, configuration, and performance optimization of a WiFi 6 (802.11ax) network. The objective is to analyze network performance, improve throughput, reduce latency, and enhance overall wireless efficiency.

## Objectives
- Study WiFi 6 (802.11ax) features
- Configure wireless network parameters
- Analyze network performance
- Optimize throughput and latency
- Improve user experience in high-density environments
